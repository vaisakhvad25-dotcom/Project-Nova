CPU & GPU Benchmark Explorer

A compact project for understanding benchmark data, spotting trends, and testing predictive recommendations.

Project overview

This project combines CPU and GPU benchmark datasets with lightweight machine-learning predictions and nearest-neighbor recommendations. The home view offers a concise summary, the data analysis view highlights benchmark distributions, and the benchmark analysis view lets you explore predictions and similar hardware matches.
